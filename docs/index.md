# Documentation Home

Welcome to the project documentation. This repository contains markdown documents that are automatically synced to Confluence on every push to `main`.

## What's Here

- **API** — REST API reference and endpoint documentation
- **Guides** — Step-by-step tutorials and how-to guides
- **Architecture** — Architecture decision records and system design docs

## How It Works

1. Edit or add a `.md` file and push to `main`
2. GitHub Actions detects the changed files
3. Each changed file is synced to Confluence, preserving the folder hierarchy
